Image Upload Demo

This demo shows the image upload functionality implemented in the Rails application. The actual upload requires the Rails server to be running.

Drop image here or click to browse

Supported formats: JPG, PNG, GIF, WebP (Max: 5MB)

Uploading...
How it works:
  • Click "Browse Files" or drag & drop an image
  • File is validated (type and size)
  • Uploaded to /api/upload endpoint
  • Returns JSON with file URL
  • Image preview is shown with remove option
Features Implemented
Products
  • Image upload in product form
  • Image display in product list (40x40px)
  • Large image in product details
  • Fallback to letter avatar
Customers
  • Image display in customer list (40x40px)
  • Image in customer details (60x60px)
  • Circular image styling
  • Fallback to letter avatar
Upload Features
  • Drag & drop support
  • File type validation
  • 5MB size limit
  • Progress indication
Security
  • CSRF token protection
  • Authentication required
  • File type restrictions
  • Unique filename generation
